# 📊 Sales & Profit Analysis Dashboard

## 🔍 Project Overview

This project analyzes retail sales data to uncover insights related to revenue, profit, customer behavior, and regional performance.

## 🛠 Tools Used

* Power BI
* Power Query
* DAX

## 📈 Key Features

* Interactive dashboard with KPIs (Sales, Profit, Orders, Profit Margin)
* Sales analysis by category and state
* Profitability analysis by sub-category
* Payment mode distribution
* Year-wise sales trends

## 💡 Key Insights

* Identified high-revenue but low-profit categories
* Detected loss-making sub-categories
* Observed seasonal sales trends
* Digital payments dominate transactions
